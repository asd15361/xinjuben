## 4. 评分异常分析

### 开局冲击（11/14）极低
两类都极低，可能原因：
1. 检测算法对"冲击"定义过严，仅识别暴力/死亡等极端开场
2. 生成提示词未强调开场冲击力
- **建议**：review `openingShockScore` 检测逻辑，看是否需要放宽阈值

### 市场匹配度男频 27 vs 女频 54
女频高出近一倍，原因：
1. 女频权力借用模式（苏晚晴→陆景琛）容易检测
2. 男频都市逆袭要素检测逻辑可能漏检
- **建议**：review 男频市场匹配检测函数，确认检测维度是否完整

### 爽点兑现均为 45
两赛道一致偏低，说明生成内容的打脸/反击时刻不够爆炸
- **建议**：增强修稿链中爽点专项提示
