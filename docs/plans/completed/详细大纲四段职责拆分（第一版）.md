# 详细大纲四段职责拆分（第一版）

## 这份文档只做一件事
- 把详细大纲四段各自到底负责什么拆死。

## 为什么必须先拆这个
- 如果四段职责不拆死，系统最容易偷懒的方式，就是把前几集、中几集、后几集直接拼一拼。
- 一旦四段职责不清，详细大纲就会重新退回“粗纲重排”。

## 四段各自的唯一职责

### 1. 开局段
- 只负责：把故事点燃。
- 必须回答：
- 主角这时候最先想守什么
- 第一个真正压下来的敌压是什么
- 主角为什么还不能马上赢
- 这一段结束时，局面被逼到哪

### 开局段不该做什么
- 不该把前三集流水账全抄一遍
- 不该提前把后面的大真相一口气讲完
- 不该写成世界观导览

## 2. 中段
- 只负责：把局面升级到退不回去。
- 必须回答：
- 对手怎么加码
- 主角怎么换打法
- 哪条关系、秘密、误判开始反咬
- 为什么到了这一段，主角已经不能回到开局状态

### 中段不该做什么
- 不该只是“继续推进”
- 不该只当开局和高潮之间的过桥
- 不该没有代价升级

## 3. 高潮段
- 只负责：逼主角亮底。
- 必须回答：
- 这一段最痛的一刀是什么
- 主角到底被逼亮了什么
- 哪层误判、真相、关系在这里一起翻面
- 为什么这是全剧最不能躲的一段

### 高潮段不该做什么
- 不该只是比中段再多打一轮
- 不该只写“冲突更大了”
- 不该没有真正的亮底和翻面

## 4. 收束段
- 只负责：收当前这轮账，并挂出下一轮。
- 必须回答：
- 眼前危机怎么压住
- 主角付掉了什么代价
- 为什么不是彻底结束
- 下一轮的更大麻烦是什么

### 收束段不该做什么
- 不该只是把最后两集再抄一遍
- 不该只有“赢了”或“输了”
- 不该没有代价和余波

## 四段之间必须形成的层级关系
- 开局：点燃
- 中段：升级
- 高潮：亮底
- 收束：收账并挂下一轮

只要四段之间没有这个层级差，就说明它还是假详纲。

## 当前硬门
- 如果一段删掉以后，整体理解几乎不受影响，这一段就是空的。
- 如果一段只是在换一种说法复述前面几集，这一段就是假的。
- 如果一段里没有主角目标、主阻力、主动作、主代价、主钩子中的大部分，这一段就不算成立。

## 当前一句话标准
- 详细大纲四段，不是四块内容，而是四次阶段职责交接。
