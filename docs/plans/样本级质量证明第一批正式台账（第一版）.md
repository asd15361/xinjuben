# 样本级质量证明第一批正式台账（第一版）

## 这次只做一件事
- 把当前最有代表性的样本，正式落成第一批样本台账。

## 为什么先做这一批
- 现在还不是要把所有样本一次堆满。
- 当前更值钱的是，先拿最能代表当前质量真相的样本，做出第一批能站住的证明。

这一批当前只收 4 类：

1. 第一板块真样本
2. 第三板块真样本

4. 表达盲区明显过线样本

这样先把：

- 真链路有没有成立
- 表达层过 / 不过怎么认

这两件最核心的事先钉死。

## 台账记录方式
- 每份样本固定按 7 项记：
- 样本名称
- 样本类型
- 样本来源
- 当前结论
- 对应质量维度
- 关键证据
- 当前定性
