# 当前主线

## 当前主线：P0 题材策略层重构

**目标：** 把男频/女频、玄幻/古言/都市/律政等题材规则从人物/骨架/剧本生成函数里解耦，建立可扩展 `GenerationStrategy` 策略层。通用引擎只管结构、合同、去重、主归属、坏 JSON 恢复和展示清洗；题材规则统一从策略包读取。

**为什么升 P0：**

- 现在玄幻男频人物小传质量已经明显变好，但这是靠真实玄幻样本修出来的。
- 项目早已有 `marketProfile.audienceLane/subgenre`，但人物和势力链路没有完整使用这套选择结果。
- 代码里仍有大量全局玄幻词：宗门、仙盟、掌门、长老、护法、魔尊血脉、亲传大弟子。
- 如果用户选择女频、都市、律政、悬疑，继续沿用这些全局规则就会串题材，产品价值会被打穿。
