**风险修复详情：**
1. **终态 task 404 竞态**（`server/src/api/routes/scripts.ts`）：原 `finally` 块立即 `runningTasks.delete(projectId)`，renderer 下次 poll 会 404。修复：延迟 60 秒删除，给 renderer 留出 poll 窗口。
2. **本地保存失败无用户提示**（`useScriptStageActions.ts`）：原 catch 块只 `console.error`。修复：添加 `setGenerationNotice` 错误提示，告诉用户"生成成功但本地保存失败，请勿刷新页面"。
