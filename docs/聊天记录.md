# 聊天记录

## 永久记忆

- 这是 AI 剧本创作软件，不是普通文本生成器。
- 正式主链固定：用户输入 → 确认信息 → `storyIntent` → 粗纲 → 人物小传 → 详细大纲 → 剧本。
- 当前工程主权固定：
  - renderer 只做展示和交互
  - server 是唯一正式业务入口
  - PocketBase 是账户与计费真相源；本地存储是内容真相源
  - IPC 只保留桌面壳职责

## 长期记忆

- 前后端分离已经完成主链收口，旧 IPC 业务链理论上应全部退场。
- 多项目共用同一个阿里云 PocketBase，必须靠 `appId` 和表前缀做隔离。
- 第三方模型通道优先；只有第三方不可用时才回退官方 DeepSeek。
- 质量门结论只认当前代码复跑结果，不认旧截图、旧口头结论。
- **架构边界已钉死**：renderer 发起与展示，main 本地只读辅助，server 正式写入 authority，PocketBase 账户/计费真相源，本地存储内容真相源。
