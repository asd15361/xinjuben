## 任务目标（本质）
- 这轮不是为了追热点，也不是为了把软件改成套路拼装器。
- 真正要解决的本质问题是：软件现在虽然已经能稳定写，但还不够像当下最能跑出来的短剧/漫剧写法。
- 如果不把爆款样本里已经被市场验证的那套“设定成交力、错位发动机、续命能力、情绪兑现”吸收到主路里，软件就容易停在“会写剧本”，而不是“会写更容易被点开、被追下去、被记住的剧本”。
