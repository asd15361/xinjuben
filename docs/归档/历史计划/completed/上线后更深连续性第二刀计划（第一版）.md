# 上线后更深连续性第二刀计划（第一版）

## 这轮只做一件事
- 在更深连续性第一刀 `前后场承接不断线` 已经正式到 `稳定通过` 以后，直接收下一张最前面的薄点：`记忆回声不丢`。

## 为什么现在必须切这张计划
- `docs/plans/上线后更深连续性第一刀阶段判断（第一版）.md` 已经正式确认：
- 第一刀 `前后场承接不断线` 已经跑出六笔同方向真实结果，当前已到 `稳定通过`

所以现在不能继续停在第一刀上反复记账。

当前更前面的本质问题是：

**剧本虽然已经能一场接一场往下走，但前面的人、前面的伤、前面的决定、前面的代价，还没被正式证明能继续在后面发声。**

## 这轮真正要解决的本质问题
- 不是再回头讲前后场承接。
- 不是先去碰硬锚点和开放钩子。
- 真正本质问题是：剧本写长以后，人物会不会像失忆一样只活在眼前一句话里。

## 为什么第二刀现在排到最前面
- `docs/plans/第二阶段第二刀连续性硬承接重排（第一版）.md` 已经定死顺序：
1. 前后场承接不断线
2. 记忆回声不丢
3. 硬锚点不失真
4. 开放钩子不烂尾

现在第一刀已经收住，下一张最前面的就是第二刀。

## 这轮第二刀只认什么
- 这轮先只认：
- 前面已经发生过的事，后面有没有继续压进人物动作、人物情绪、人物选择和当场代价里。

更直白一点：
- 不能只是“提过一句以前发生过”。
- 必须看见“以前那一下，到现在还在影响人怎么做、怎么怕、怎么赌”。

## 为什么先这么认
- 因为记忆回声不是回忆杀。
- 真正值钱的，不是角色嘴上想起过去。
- 真正值钱的是：过去发生过的事，还在继续推今天这一场。

## 这轮验收标准
- 要把 `记忆回声不丢` 压成当前唯一现行口径
- 要能正式回答：
- 这条第二刀现在推进到了哪一步
- 它有没有跑出第一笔真实结果
- 它离 `稳定通过` 还有多远

## 当前判断
- 当前不该平均发力到整层连续性。
- 当前也不该跳去第三刀和第四刀。
- 当前最前面的主问题，就是先把 `记忆回声不丢` 压住。
