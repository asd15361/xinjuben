# 创作全链路批次统一与逐集细纲计划

更新时间：2026-03-20

## 一句话结论

参考爆款短剧数据（均字670-760/集，均场1.5-2.3场），重新设计四层批次统一和逐集细纲结构。

---

## 爆款样本数据依据

| 指标 | 爆款样本中位数 | 设计目标 |
|------|---------------|---------|
| 均字/集 | 670-760字 | 500-800字 |
| 均场/集 | 1.5-2.3场 | 2-3场 |
| 每场字数 | 300-400字 | 150-300字 |

---

## 四层批次设计

```
粗纲批次 = 10集
    ↓ 联动
人物小传批次 = 最多5个/批
    ↓
详细大纲批次 = 10集（逐集生成，每集有逐集细纲）
    ↓ 每5集
剧本批次 = 5集（A=前情提要 + B=逐集细纲）
```

---

## 各环节批次详细设计

### 1. 粗纲（10集批次）

| 项目 | 设计 |
|------|------|
| 批次大小 | 10集/批 |
| 输出字段 | episode, title, coreConflict, keyEvents(3-5条), cliffhanger |
| 字数目标 | 150-200字/集 |
| 人物联动 | 每粗纲批次 → 生成人物小传批次 |

### 2. 人物小传（最多5个/批）

| 项目 | 设计 |
|------|------|
| 每批数量 | 最多5个 |
| 超过5个 | 分批生成 |
| 批次关联 | 每批对应粗纲批次 |

### 3. 详细大纲（核心新增逐集细纲）

| 项目 | 设计 |
|------|------|
| 生成方式 | 逐集串行 |
| 每集场数 | 2-3场 |
| 逐集细纲 | 每场1段100-200字描述 |
| 每集总字数 | 300-500字（不含逐集细纲额外开销） |

#### SceneBeat 场次结构

```typescript
interface SceneBeat {
  sceneNo: number      // 第几场
  location: string     // 地点
  timeOfDay: string    // 日/夜
  setup: string         // 本场背景（50字）
  tension: string       // 本场张力目标（50字）
  hookEnd: string       // 场尾钩子（50字）
}

interface EpisodeOutlineEnhanced {
  episode: number
  title: string
  content: string           // 阶段总述
  sceneByScene: SceneBeat[] // 逐集细纲（2-3场）
}
```

### 4. 剧本（5集批次，A+B喂法）

| 项目 | 设计 |
|------|------|
| 批次大小 | 5集/批 |
| A=前情提要 | 600+600+500 = 1700字 |
| B=逐集细纲 | 本批次5集×2-3场×150字 ≈ 1500-2250字 |
| 每集字数 | 500-800字 |
| 字数监控 | 超900字预警 |

---

## 剧本格式（参考样本）

```
第1集
1-1日内灵堂
人物：司念，司辛，张翠梅
△司念的脚动了一下，正被准备鞠躬的司辛看见。
司辛：（惊慌拉住张翠梅）她好像动了一下。
张翠梅：（不耐烦）你看错了。
```

格式规则：
- 场次标题：场号-日内/外+地点
- △开头=动作
- 角色：(情绪)对白
- 括号=情绪

## 改动清单

### 类型改动

| 文件 | 改动 |
|------|------|
| `EpisodeOutline` 类型 | 增加 `sceneByScene: SceneBeat[]` |
| 新增 `SceneBeat` 类型 | 定义场次结构 |
| 新增 `RoughOutlineBlock` | 粗纲批次结构 |
| 新增 `CharacterBatch` | 人物小传批次 |
| 新增 `ScriptBatch` | 剧本批次 |

### 逻辑改动

| 环节 | 改动 |
|------|------|
| 粗纲生成 | 改为10集一批 |
| 人物小传生成 | 改为最多5个一批 |
| 详细大纲生成 | 增加逐集细纲输出，每集2-3场 |
| 剧本prompt | 改为只吃逐集细纲作为B输入 |

## 执行步骤

### Step 1: 定义新类型
- 新增 SceneBeat 类型
- 增强 EpisodeOutline
- 新增批次类型

### Step 2: 粗纲生成改为10集批次
- 修改 outline-generator.ts batchSize=10

### Step 3: 人物小传改为分批（最多5个）
- 修改 characterProfilePromptService.ts

### Step 4: 详细大纲增加逐集细纲
- 修改 generateDetailedOutline 输出
- 每集生成2-3个 SceneBeat

### Step 5: 剧本只吃逐集细纲
- 修改 create-script-generation-prompt.ts
- B输入改为逐集细纲

---

## 禁止事项
- 禁止兼容字段回流
- 禁止逐集细纲为空生成剧本
- 禁止每集超过3场或低于2场（除非剧情需要）

---

## 验收标准
- 每集2-3场
- 每集500-800字
- 逐集细纲每场100-200字
- 剧本只从逐集细纲取B输入
