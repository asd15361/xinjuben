# 人物活性专项样本计划（第一版）

更新时间：2026-03-16 09:03

## 这份计划解决什么本质问题
- 不是再证明“人物活性重要”。
- 真正要解决的是：人物活性这条虽然已经有稳定通过结果，但还没并成总样本链里的现行专项样本组。

## 先说总判断
- 人物活性当前不是没结果。
- 当前缺的是：
  - 现行专项样本计划
  - 现行专项样本台账
  - 并回总样本链的正式判断

## 这组专项样本要证明什么
- 要证明的不是“人物资料更完整”。
- 要证明的是：
  - 人物不会把试探直接说破
  - 人物会按自己的人去试、去藏、去逼
  - 行为前因后果能成立
  - 交锋像人物自己活出来，不像作者代说

## 当前样本来源
- 当前专项样本直接用这条稳定通过链做第一组正式来源：
  - `上线后人物活性第一刀稳定通过判断（第一版）.md`
  - `上线后人物活性第一刀稳定通过第一轮结果（第一版）.md`
  - `上线后人物活性第一刀稳定通过第二轮结果（第一版）.md`
  - `上线后人物活性第一刀下一轮执行结果（第一版）.md`

## 这组专项样本固定怎么记
- 每笔样本都固定记 5 件事：
1. 修前原句
2. 修后原句
3. 当前主要改善点
4. 系统复述是否继续同方向
5. 当前定性

## 当前只认哪些人物活性证据
- 只认这些能看见的东西：
  - 目的不直说
  - 试探不自报
  - 亮底问题改成留口问题
  - 解释挪到动作后面

## 当前验收标准
- 这组专项样本要成立，至少同时满足：
1. 连续多轮同方向成立
2. 问题类型一直没换
3. 顺序一直没乱
4. 新结果不是反复改旧句，而是新的真实输出还能继续同方向成立

## 当前收口顺序
### 第一阶段
- 先把人物活性稳定通过链正式并成专项台账。

### 第二阶段
- 再给这组样本写专项判断。

### 第三阶段
- 最后把专项判断并回总样本链。

## 最后结论
- 人物活性这条现在最缺的不是结果，而是“稳定通过结果还没入现行总账”。
- 这张计划就是把这条旧稳定结果正式变成总样本链的一部分。
