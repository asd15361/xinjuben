# 通用文档母版说明

这一套文件只有一个用途：
**直接复制进任意项目，让人和 AI 按同一套规则协作。**

## 怎么用
- 把整套文件复制到新项目根目录。
- 新会话先看 `AGENTS.md`。
- 正式任务开始前更新 `docs/当前工作区/active-task（当前任务卡）.md`。
- 重要阶段结论写进 `4.worklog.md`。
- 单独大任务写进 `docs/plans/`。

## 这套母版故意不写什么
- 不写某个具体项目的业务规则。
- 不写某个具体项目的目录名、组件名、服务名。
- 不写任何敏感信息。

## 需要项目自己补的部分
- 当前项目目标
- 当前项目专属流程边界
- 当前项目任务卡内容
- 当前项目阶段记录和计划
