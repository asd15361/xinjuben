# 最薄表达盲区第一批对照样本卡（第一版）

## 一、这份文档解决什么问题

这份文档只回答一件事：

**按已经定好的样本卡标准，先把最薄表达盲区的第一批最小对照样本真正落出来。**

不是一下子堆很多。  
而是先做最小一批，证明这条路能走。

---

## 二、先说结论

第一批当前先做 4 组，一组盲区一张“明显不过”卡：

1. 直接把试探说破
2. 只剩信息，没有人物当下压强
3. 句式太像总结句
4. 停顿和重心落不住

这一版先不追求“过线卡”一起写全，  
先把“明显不过卡”立起来，先把坏法钉死。

---

## 三、样本卡 1：直接把试探说破

### 1. 属于哪个盲区

- `直接把试探说破`

### 2. 原句是什么

- “你别误会，我现在就是想试试你到底还站不站在我这边。”

### 3. 为什么不过

- 本来该藏在来回里的试探，被人物自己一句话说穿了
- 戏剧动作没展开，底牌先亮了

### 4. 对应正式判断点是什么

- `说明味过重 -> 直说这场要干什么 -> 直接把试探说破`

### 5. 如果算过，过的样子应该是什么

- 人物不直接说“我在试你”
- 而是在绕、逼、探、等回应里把试探做出来

### 6. 当前标签

- `明显不过`

---

## 四、样本卡 2：只剩信息，没有人物当下压强

### 1. 属于哪个盲区

- `只剩信息，没有人物当下压强`

### 2. 原句是什么

- “公司已经决定撤掉你这个项目负责人，通知今天下午会正式发。”

### 3. 为什么不过

- 信息本身清楚，但人物当下情绪、关系位置、压力都没出来
- 换谁说都差不多，只像功能通报

### 4. 对应正式判断点是什么

- `说明味过重 -> 缺少人物当下状态 -> 只剩信息，没有人物当下压强`

### 5. 如果算过，过的样子应该是什么

- 同样是通知，也要带着人物此刻的难堪、压抑、试探、站位或逼迫感

### 6. 当前标签

- `明显不过`

---

## 五、样本卡 3：句式太像总结句

### 1. 属于哪个盲区

- `句式太像总结句`

### 2. 原句是什么

- “我们之间当前最大的症结，本质上是长期失衡的信任结构。”

### 3. 为什么不过

- 这句话像结论和分析，不像当场的人会这样接话
- 写出来像成立，嘴里说出来像报告

### 4. 对应正式判断点是什么

- `台词拗口 -> 太像写出来的话 -> 句式太像总结句`

### 5. 如果算过，过的样子应该是什么

- 同样的意思，也该落成当场会说的话，而不是像总结报告

### 6. 当前标签

- `明显不过`

---

## 六、样本卡 4：停顿和重心落不住

### 1. 属于哪个盲区

- `停顿和重心落不住`

### 2. 原句是什么

- “如果你现在仍然坚持以这种并不成熟但又过度强硬的方式继续处理这件事，那我们之后就真的很难再谈。”

### 3. 为什么不过

- 句子过长，前后重心挤在一起
- 不知道该停在哪，也不知道重音该落哪
- 读起来明显发卡

### 4. 对应正式判断点是什么

- `台词拗口 -> 节奏发卡 -> 停顿和重心落不住`

### 5. 如果算过，过的样子应该是什么

- 该拆开的拆开
- 重心清楚，停顿自然，嘴里说得出去

### 6. 当前标签

- `明显不过`

终于有统一底板可以对。
