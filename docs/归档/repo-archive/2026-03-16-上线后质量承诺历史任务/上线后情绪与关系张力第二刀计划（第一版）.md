# 上线后情绪与关系张力第二刀计划（第一版）

## 这轮只做一件事
- 在第一刀已经把“只剩信息，没有人物当下压强”接回创作主路以后，继续收下一刀：`谁说都行句`。

## 为什么现在必须切这张计划
- 第一刀已经解决了更前面的本质问题：句子不能再只剩信息功能。
- 但现有旧顺序里，下一刀本来就不是乱跳，而是继续往下收：
- 先拆纯通报句
- 再拆谁说都行句
- 再拆没站位句
- 最后拆没情绪压强句

所以现在正确动作不是回头重讲第一刀，而是继续推进第二刀。

## 这轮真正要解决的本质问题
- 不是“再多加一点情绪词”。
- 不是“再多提醒一次人物名字”。
- 真正本质问题是：

**有些对白虽然不再只是通报信息了，但换给另一个人说也成立，说明人物还没有真正长在这句话上。**

## 为什么这还是创作问题
- 这不是系统识别不到名字。
- 这不是字段没传到。
- 这是“这句话为什么非得这个人现在说”还没写出来。
- 所以这条问题仍然必须靠创作合同去收，不能靠 gate、命中词校验或补名单去装作解决。

## 这轮落点只认哪里
- 主生成说明
- 阶段引导说明
- 修补说明

## 这轮最小动作
1. 写对白前先问：这句话换另一个人说还成立吗。
2. 如果成立，就继续改，直到能听出这个人的位置、口气和处境。
3. 不准只靠补情绪词冒充人物口气。

## 验收标准
- 能正式回答：这条问题已经回到创作合同，不再靠系统门解决。
- 能正式回答：现行说明已经把“谁说都行就重写”变成硬动作。
- 能正式回答：这条问题和第一刀已经分开，不再混成一句大空话。

## 当前判断
- 第一刀已完成第一轮落地。
- 当前下一刀正式切到：`谁说都行句`。
