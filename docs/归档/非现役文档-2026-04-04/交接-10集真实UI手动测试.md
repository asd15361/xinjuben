# 交接文档：10集真实UI手动测试

## 一、我做了什么

本轮完成了一个**系统性链路重构**，修了3个核心问题：

### 1. 胖集根因——评估口径错位
- **原问题**：`char-count-agent` 用 `screenplayScenes[].body` 总和评估字数（~1700字），guard 合同检查用 `screenplay` 字段（A+D+E拼接，~2000字）。7个胖集全部是这种情况，agent 以为没超线，guard 判定超了。
- **修复**：`countSceneChars()` 口径完全对齐 guard（直接用 `getScreenplay()`）；胖稿 prompt 从"保冲突不压字数"改成"密度分配"指令。

### 2. 链路超时——repair_batch 串行阻塞
- **原问题**：`waitForRepairBatch` 默认 `true`（同步阻塞），batch跑完30集后再串行追加最多60次API调用，拖爆50分钟超时。
- **修复**：`waitForRepairBatch` 默认改 `false`，batch完成立即返回保存链，repair在后台异步跑。

### 3. 3个质量Agent从未生效
- **原问题**：`enableContentRepair` 链路断了3处，从未传到 `repairRunner`。
- **修复**：`enableContentRepair` 改默认 `true` 并打通链路；guard repair 和 content repair 完全并行，不再要求"guard全过才进content repair"。

### 改动文件清单
- `src/main/application/script-generation/start-script-generation.ts`

## 四、参考代码

下面是 `manual-10ep-playwright-test.mjs` 的参考实现（核心部分）：

```javascript
import { _electron as electron } from 'playwright'

async function main() {
  const repoRoot = 'D:/project/xinjuben'
  const mainEntry = `${repoRoot}/out/main/index.js`

  const app = await electron.launch({
    args: [mainEntry],
    env: { ...process.env, E2E_CASE_ID: `manual-10ep-${Date.now()}` }
  })

  const page = await app.firstWindow()
  await page.waitForLoadState('domcontentloaded', { timeout: 20000 })
  await page.setViewportSize({ width: 1440, height: 960 })
  await page.waitForTimeout(1500)

  // 截图：首页
  await page.screenshot({ path: `out/e2e/01-home.png`, fullPage: true })

  // 创建项目
  await page.click('input[placeholder="新项目名称..."]')
  await page.fill('input[placeholder="新项目名称..."]', `手动10集测试-${Date.now()}`)
  await page.click('button:has-text("快速创建")')
  await page.waitForTimeout(2000)
  await page.screenshot({ path: `out/e2e/02-project-created.png`, fullPage: true })

  // === 工序流程 ===
  // 1. 聊天 → 确认信息
  // 2. 粗纲 → 确认进入人物
  // 3. 人物小传 → 确认进入详细大纲
  // 4. 详细大纲 → 确认进入剧本
  // 5. 剧本页面 → 设置10集 → 点击生成

  // 关键：记录每个按钮的 disabled 状态
  const buttons = [
    '我聊完了，确定信息',
    '生成第一版粗纲和人物',
    '确认：进入人物小传',
    '确认：进入详细大纲',
    '生成这一版详细大纲',
    '剧本定稿',
    '现在开始写剧本'
  ]

  for (const btnText of buttons) {
    const btn = await page.$(`button:has-text("${btnText}")`)
    if (btn) {
      const isDisabled = await btn.evaluate(el => el.disabled)
      console.log(`按钮[${btnText}]: ${isDisabled ? '灰掉(disabled)' : '可点击'}`)
      if (!isDisabled) {
        await btn.click()
        await page.waitForTimeout(3000)
        await page.screenshot({ path: `out/e2e/03-${btnText}.png`, fullPage: true })
      }
    }
  }

  // 设置集数为10
  const episodeInput = await page.$('input[type="number"], input[placeholder*="集"]')
  if (episodeInput) {
    await episodeInput.fill('10')
    await page.waitForTimeout(500)
    await page.screenshot({ path: `out/e2e/04-episode-set.png`, fullPage: true })
  }

  // 点击生成剧本
  const genBtn = await page.$('button:has-text("现在开始写剧本")')
  if (genBtn) {
    const isDisabled = await genBtn.evaluate(el => el.disabled)
    console.log(`[现在开始写剧本] 按钮: ${isDisabled ? '灰掉(disabled)' : '可点击'}`)
    if (!isDisabled) {
      await genBtn.click()
      console.log('开始生成，等待进度...')

      let lastCount = 0
      while (true) {
        await page.waitForTimeout(30000) // 每30秒检查一次
        await page.screenshot({ path: `out/e2e/progress-${lastCount}.png`, fullPage: true })

        // 检测错误弹窗
        const errorModal = await page.$('[role="alert"], .error, .ant-modal-error')
        if (errorModal) {
          const errText = await errorModal.textContent()
          console.log('发现错误弹窗:', errText)
          await page.screenshot({ path: `out/e2e/ERROR.png`, fullPage: true })
          break
        }

        // 检查是否完成
        const completed = await page.evaluate(() => {
          // 找已完成集数的显示
          const els = document.querySelectorAll('[class*="completed"], [class*="done"]')
          return els.length
        })

        if (completed > lastCount) {
          lastCount = completed
          console.log(`已完成 ${lastCount} 集...`)
        }

        if (lastCount >= 10) {
          console.log('10集全部生成完毕！')
          await page.screenshot({ path: `out/e2e/05-complete.png`, fullPage: true })
          break
        }

        // 超时保护（90分钟）
        // ...
      }
    }
  }

  await app.close()
}

main().catch(console.error)
```
