# 短剧爆款黄金铁律 · 2700部爆款总结 (2026-04-11)

## 一、 短剧黄金铁律 · 10条 (系统基石)
1. **开局即高潮**：开局30秒必须死人、丢官、退婚、被夺、被踩、被冤枉。
2. **极速反转**：1分钟必须出现第一次反转或第一次打脸。
3. **单集结构**：每集只干一件事：施压 → 反击 → 留钩子。
4. **主角风骨**：主角可以弱，但绝对不能窝囊、不能哭、不能崩溃。
5. **高智反派**：反派可以坏，但不能靠吼，要靠权力、规则、布局。
6. **节奏爽点**：每5集必须来一次大爽点（当众打脸/身份揭晓/碾压）。
7. **对白精简**：所有解释性台词必须压缩在8秒内，不许讲世界观。
8. **成本意识**：场景越少越值钱，最好3-5个场景拍完整部剧。
9. **金句驱动**：金句=传播力，没有金句=没有流量。
10. **钩子致命**：结尾必须停在最爽的前0.1秒，强制看下一集。

## 二、 单集标准结构 · 固定三段式
*   **第1幕：施压 (0–20秒)**
    *   反派当众羞辱；拿人质、拿把柄、拿规则压人；把主角逼到无路可退。
    *   *观众情绪：愤怒、憋屈。*
*   **第2幕：反转 (20–45秒)**
    *   主角拿出证据/底牌/后手；反派脸色突变；旁观者震惊。
    *   *观众情绪：爽、解气。*
*   **第3幕：钩子 (45–60秒)**
    *   新危机、大反派、大秘密；最后一句台词扎心，强制留客。

## 三、 主角行为规范 (硬约束)
*   可以忍，但不能怕；可以装弱，但不能真弱。
*   退让必须是为了布局；受伤不能崩溃。
*   眼神冷、定、稳；受辱必须在同一集讨回来。
*   绝不解释，只用实力/证据/底牌说话。

## 四、 反派行为规范 (高智商压迫)
*   **禁止**：大吼大叫、无脑骂街、强行降智、重复栽赃。
*   **必须**：用规则杀人、用权位压人、用利益分化、微笑着把人逼死。
